## :rocket: Installation

Clone the repository into your machine and install the dependencies:

```bash
npm install
```

Add your firebase app credentials in this file:

```bash
./src/firebase.js
```

Run the following command to run the server, see the result in http://localhost:3000/:

```bash
npm start
```
